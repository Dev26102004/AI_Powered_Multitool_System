# 🤖 AI-Powered Multi-Tool System

## 📌 Overview
This project is an **AI-powered multi-tool assistant** that combines Large Language Models (LLMs) with multiple tools to solve real-world tasks.

The system is designed to intelligently decide when to use:
- Web search for real-time information
- A calculator for mathematical problems
- Retrieval-Augmented Generation (RAG) for document-based queries

Currently, the project includes a working **RAG pipeline prototype** and is being extended into a full **agent-based AI system**.

---

## 🚀 Features

- 🔍 Web Search Integration (real-time information)
- 🧮 Advanced Calculator for mathematical reasoning
- 📄 RAG-based Document Question Answering
- 🧠 LLM-powered reasoning and response generation
- 🔗 Multi-tool integration with intelligent routing (in progress)

---

## 🧠 System Architecture

### Current Implementation:
1. Load documents (PDF/Text)
2. Split into chunks
3. Generate embeddings using HuggingFace
4. Store embeddings in FAISS
5. Retrieve relevant data based on query
6. Generate answer using LLM

### In Progress:
- Tool-based AI agent
- Query routing system
- Multi-tool execution pipeline
